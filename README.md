# Proyecto-Final-Lenguajes

Version: C++ 14

Integrantes: Andrés Julián Gil Nagles

El programa recibe una oracion y la clasifica por su sujeto, verbo y predicado para luego verificar si esta escrita de la forma SUJETO + VERBO + PREDICADO dando error cuando el sujeto y el verbo se escriben después del predicado, cuando una oracion tiene 2 o mas verbos y con oraciones con predicados con núcleo copulativo compuesto
